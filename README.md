# AI-Driven Agriculture Yield Prediction & Market Insights

A modern Flask web application that provides AI-powered crop yield predictions and market insights for farmers and agricultural stakeholders.

## Features

- **Home Page**: Modern hero section with animated navigation
- **Problem Identification**: Visual presentation of agricultural challenges
- **Yield Prediction**: AI-powered crop yield prediction with interactive charts
- **Market Insights**: Real-time market data and trend analysis
- **Responsive Design**: Mobile-friendly with modern glassmorphism UI

## Tech Stack

- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Charts**: Chart.js
- **Styling**: Custom CSS with animations and responsive design
- **Fonts**: Google Fonts (Poppins)
- **Icons**: Font Awesome

## Installation

1. Install Python dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser and navigate to `http://localhost:5000`
